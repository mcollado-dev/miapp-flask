Funciones Disponibles

Esta sección describe las principales funcionalidades implementadas en la aplicación Flask, junto con sus objetivos y comportamiento dentro del sistema.

Gestión de Usuarios

Permite crear, consultar y listar los usuarios almacenados en la base de datos SQLite. Cada usuario tiene un nombre, un correo electrónico y un rol asignado.

Generación de Estadísticas

Utiliza la librería Matplotlib para generar gráficos dinámicos en formato base64, mostrando la distribución de roles entre los usuarios registrados.

Exportación y Análisis de Datos

El sistema está preparado para incorporar funciones de exportación de datos (por ejemplo, a CSV o PDF) y análisis estadístico avanzado de los registros almacenados.

Configuración y Personalización

La aplicación incluye opciones configurables mediante variables de entorno y archivos de configuración para adaptar el despliegue a distintos entornos (desarrollo, pruebas o producción).

Despliegue Automatizado

Integración con Jenkins y Docker para construir, analizar y desplegar automáticamente la aplicación tras cada commit en el repositorio.
